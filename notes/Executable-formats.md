### Executable Formats

An executable format is the file created by the complier and linker. Lets say, you want to your OS to open a game that you created, when you execute it, the complier will compile it to a executable format. Now, you can link your OS to the executable file so that the game loads.

### Types of executable files

| Format | Used By |
|:----------:|:----------:|
| ELF    | Linux, Unix, BSD    | 
| PE    | Windows.exe     |
