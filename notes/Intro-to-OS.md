### What is an Operating System

- A software that controls the operation of a computer system and its resources. 
- Capable of loading and executing user programs providing, standardized (hardware-independant) input / output interface for them.

### Major Functions

- Managing Memory
- Scheduling and multiplexing proccess and threads
- Launching and closing user programs dynamically
- Providing a basic user interface and application programmer interface

NOTE: Not all operating system provide these functions

### What is a kernel

The kernel is the brain. It enables any programs to execute. It handles events by hardware( called interupts) and software (called system calls), and manages access to resources.

### What is a shell

A special program that is usually integrated with the OS distribution, that offers a way so that humans can interact with computers. It allows user to select a program to be started and more. For example: local storage like listing the directory, files inside it and more. 

### GUI (Graphical User Interface)

The Graphical user Interface (GUI) is the most viable part of the Operating System. It catches user inputs events(Keyboard, mouse, etc.) and dispatch them to proper use. It should update screen contents so that it feels natural and more.
